IBM ACD/IML is a powerful set of tools that allows researchers to automate the exploration and discovery of medical concepts in big corpus of documents. Build on top of IBM Watson's natural language understanding model, this tool offers a deep understanding of each document, to extract all concepts and their relationships to give the researchers true insights, important trends, and new discoveries. This web application uses PubMed, CTGOV and Medline corpora which are already pre-integrated, however, the user can define his own corpus to use the pre-trained NPL model on his own data.

Advanced Search

Search for specific concepts to get advanced statistics such as historical term frequency and co-occurrences.

Knowledge Network

Discover relationships between concepts and documents using a network graph.

Explore your results

Get more relevant results and discover complex relationships between concepts using Insights for Medical Literature API.
